I created responsive Portfolio website for myself using HTML and CSS.

Tags used: 
Header: Top part of page to create navigation bar

Nav: For fixed navbar

img: For rounded profile image

H1, H2, H3, H4: For headings

Section: to divide page in sections

div: to create container to apply stylings

p: for paragraph

a: for hyperlinks

ul,li: for unordered list

footer: for lowest part of page

mailto and tel is used for contact

@media screen is used for apply responsive properties

position property is used for positioning elements

float and overflow properties are used

column layout structured is used to arrange sections

image gallery is created for certificates and zoom in on hover.

**Font-Awsome Icons are used for social links using <i> tag
